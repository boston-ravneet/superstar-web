GUIDELINE 2.1 — THIRD-PARTY AI
Yes, we use Google Gemini (Google Cloud) only when the user taps "Create my page" or "Rebuild my page" — for profile layout/copy generation and image moderation. We do not offer AI chat.
Data sent to Gemini (only on explicit build): user-selected photos, bio, optional design notes, optional social handles (text), display name, public handle. We do NOT send sign-in tokens, passwords, payment data, or device IDs.
Consent: Sign-in requires accepting Terms & Privacy (checkbox). Existing users without current terms see an Accept Terms screen; AI/build is blocked until accepted.

GUIDELINE 2.1(b) — BUSINESS MODEL
No subscriptions, IAP, or payments. App is 100% free. All features (handle, photos, AI page, publish, analytics) are free. "Sponsored video" during generation is placeholder loading UI only, not ads or monetization. Sign in with Apple or Google — no fee.

GUIDELINE 2.1(a) — CREATE MY PAGE SPINNER
Fixed in build 11: request timeouts, clearer errors, mobile multipart upload fix, progress labels on build screen, removed "Upgrade later" copy. Test: Sign in → claim handle → add bio/photos → Create my page.
